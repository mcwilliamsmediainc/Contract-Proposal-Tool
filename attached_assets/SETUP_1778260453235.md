# McWilliams Media Audit Tool — Replit Setup Guide

## Files in This Package

```
server/
  db.js           ← Replit DB data layer (leads, proposals)
  auditRoutes.js  ← Express API endpoints
  scanner.js      ← Website scanner + Claude AI interpreter
  email.js        ← Email handler (plug into your existing setup)
  pdfReport.js    ← Branded PDF generator

src/audit/
  useAudit.js     ← React hook for all API calls
  AuditTool.jsx   ← Main React component (full sales page)
```

---

## Step 1 — Install Dependencies

Open the Replit Shell and run:

```bash
npm install @replit/database @anthropic-ai/sdk node-fetch pdfkit uuid
```

---

## Step 2 — Add Replit Secrets

Click the **Secrets** tab (lock icon) and add:

| Key | Where to Get It |
|-----|----------------|
| `ANTHROPIC_API_KEY` | console.anthropic.com → API Keys |
| `PAGESPEED_API_KEY` | console.cloud.google.com → Enable "PageSpeed Insights API" → Create Key (free) |
| `GOOGLE_PLACES_API_KEY` | Same Google Cloud Console → Enable "Places API" → Create Key (free) — or use same key as PageSpeed |
| `TEAM_API_KEY` | Make up any strong random string — protects the /dashboard endpoint |

---

## Step 3 — Wire Into Your Existing server.js

Find your main Express server file (server.js or index.js) and add exactly these two lines:

```javascript
// Add near the top with your other requires:
const auditRoutes = require('./server/auditRoutes');

// Add after your other app.use() calls:
app.use('/api/audit', auditRoutes);
```

Do not change anything else.

---

## Step 4 — Plug In Your Existing Email Setup

Open `server/email.js` and find the three `── PLUG IN HERE ──` blocks.

Replace each one with your existing email method. Most common on Replit is Nodemailer:

```javascript
await transporter.sendMail({
  from:    `McWilliams Media <info@mcwilliamsmedia.com>`,
  to:      lead.email,
  cc:      'info@mcwilliamsmedia.com',
  subject,
  html:    htmlBody,
  text:    textBody,
  attachments: [{
    filename:    `McWilliams-Media-Audit-${Date.now()}.pdf`,
    content:     pdfBuffer,
    contentType: 'application/pdf',
  }],
});
```

If you use SendGrid, Resend, or another service, the variables are the same — just swap the method.

---

## Step 5 — Add the Frontend Route

In your React router file (App.jsx or wherever routes are defined):

```jsx
import AuditTool from './audit/AuditTool';

// Inside your <Routes>:
<Route path="/audit" element={<AuditTool />} />
```

## Step 6 — Test End to End

Go to `mcwclients.com/audit` and run through the full flow:

1. Enter a real URL and city — watch the scan animation
2. Confirm real scores come back
3. Enter a test email — confirm PDF arrives in your inbox
4. Complete the qualification form
5. Click "Request a Proposal" — confirm team gets the alert

## Step 7 — Optional: Set Up Subdomain

Once working, point `audit.mcwilliamsmedia.com` to your Replit app.

In Replit: Settings → Custom Domains → add `audit.mcwilliamsmedia.com`

This keeps prospect-facing URLs clean and separate from the client portal.

## API Endpoints Reference

| Method | Path | Description |
|--------|------|-------------|
| POST | /api/audit/create | Create lead, returns leadId |
| POST | /api/audit/scan | Run real scan, returns 7 unlocked scores |
| POST | /api/audit/capture | Save email, unlock Social + AI scores |
| POST | /api/audit/qualify | Save budget/goal, create proposal shell |
| GET | /api/audit/request-proposal?leadId= | Proposal request from email link |
| POST | /api/audit/request-proposal | Proposal request from in-app button |
| GET | /api/audit/dashboard | Team stats (requires x-api-key header) |
| GET | /api/audit/lead/:id | Single lead + proposal (requires x-api-key) |
| PATCH | /api/audit/proposal/:id/status | Update proposal status (requires x-api-key) |

---

## Replit DB Key Structure

All audit keys are prefixed — no collision with existing data:

```
lead:{uuid}              Full lead object
lead:email:{email}       Reverse lookup: email → leadId
lead:list                JSON array of all lead IDs
lead:pending             JSON array of leads awaiting proposal
proposal:{uuid}          Full proposal shell
```
